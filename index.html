AirGolf - Physics Game

A 3D physics game with a mix of golf and air hockey. With the low friction dynamics of air hockey and interesting physics puzzles. I created the first version in Unreal Engine but latter ported it to Unity. All design, assets, 3D models, code, animations and shaders are written by me.

Try it out! Don't forget to buff the graphics if you got a beefy graphics card. Windows Linux Mac OSX

Looptime - Temporal Game Engine

Looptime is a prototype 3D multiplayer game engine built in HTML5, WebGL, JavaScript and GoLang. It features free-form jumps in time, grandfather paradoxes and other similar temporal shenanigans. With network latency compensation and matchmaking. Built together with my friend and classmate Viktor Palmkvist.

MetaTV - Electronic Billboards System

News a billboard system for the pub and lunch area for the Computer Science students at KTH Royal Institute of Technology. It is running on a Linux server and 3 Raspberry Pi 2's and large screens. Written with Node.js, WebSockets and Meteor.js. Upcoming events, funny pictures, drink menus for the pub, gif's, YouTube videos and entire websites. There is full version control, tag filtering, authentication and automatic removal of old slides. MetaTV is a project with 16 contributors, I wrote and designed the original version of this project and have since maintained and reviewed all code in pull requests on Github.

Ordkraft - Readability Experiment

I wanted to test if readability of regular text could be improved in the same way code is by highlighting structures in text with colours. So I wrote a syntax highlighter for text, colouring suffixes, vowels and numbers. It failed and is maybe even harder to read but it is a cool experiment.

Auto Wiki Info Box - Text extraction AI

As a large school project, I together with Daniil Pintjuk, Viktor Palmkvist and Marcel Enguehard built a text-information-extractor. It reads Wikipedia articles and builds the infoboxes on the side with information such as date of birth and residence for people-articles, but it is general and can be made to work with any article type. It trains Hidden Markov Models with data from articles that already have info-boxes and learns what sentence structure usually prefixes and postfixes the information we are searching for. It is written in C++ and Python.

Github

Amensty - Permission System

I created the GUI for an application called Amnesty, a sophisticated relational database for permission tracking at the Computer Science student chapter at KTH Royal Institute of Technology. The system tracks students, what roles they have, what permissions the roles have and what position each role has in the organisation. It can be viewed from multiple angles depending on the action and the front end reflects these actions. As the system is very complex, the documentation had to be included into the GUI with warnings, explanations and footnotes. Amnesty is written in Ruby on Rails.
