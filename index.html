AirGolf - Physics Game

3D physics game with a mix of golf and airhockey. With the low friction dynamics of air hockey and interesting physics puzzles. Written in Unreal Engine 4. All assets, 3d models, animations and shaders are written by me.

Looptime - Temporal Game Engine

Looptime is a prototype 3D multiplayer game engine built in HTML5, WebGL, JavaScript and GoLang. It features free-form jumps in time, grandfather paradoxes and other similar temporal shenanigans. With network latency compensation and matchmaking server.

Built together with my friend and classmate Viktor Palmkvist. Github

MetaTV - Electronic Billboards System

News billboard system for the pub and lunch area for the computer science students at KTH. Running on a Linux server and 3 Raspberry pi 2's and large screens. Written with Node.js, websockets and Meteor.js. Upcoming events, funny pictures, drink menus for the pub, gif's, YouTube videos and entire websites. There is full version control, tag filtering, authentication and automatic removal of old slides. I wrote the original version of this project and have since maintained and reviewed all code in pull requests on Github. All code can be found on Github
